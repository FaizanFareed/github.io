Sliding Window Counter Rate Limiter
To protect API from DDoS attack, avoid spikes in traffic, running API efficiently and for API scalability rate limiter component is essential. Python sliding window counter API rate limiter using Redis.
Python Redis MIT license


Read more

Compare to other rate limit algorithms sliding window counter is

  1. Avoid spikes (Smooth our brusts)
  2. Memory effiecint

Rate limit strategies allowed in this algorithm

  1. N requests per second.
  2. N requests per minute
  3. N requests per hour

Rate limit HTTP headers allowed in this algorithm

  1. X-RateLimit-Limit
  2. X_RateLimit_Remaining
  3. X-RateLimit-Reset
  4. Retry-after
Optional parameters can be passed to rate limiter when first time rate limiter initialized. These parameters are used for performance and reduce memory usage.
